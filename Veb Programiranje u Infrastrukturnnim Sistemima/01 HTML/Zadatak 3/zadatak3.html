Web čitači

Web čitač je aplikacija koja je namenjena prikazu sadržaja web stranica. Stranice se mogu dobaviti kako iz fajl-sistema, tako i putem TCP/IP mreže. Protokol kojim komuniciraju web čitač i web server je HTTP protokol. Danas su najpoznatiji web čitači:

  1. Microsoft Internet Explorer (https://microsoft.com/ie)
  2. Mozila (https://mozilla.org)
  3. Opera (https://opera.com)

Obratite pažnju da su svi nazivi web čitača u gornjoj listi linkovi koji upućuju na adrese navedene u zagradi!
